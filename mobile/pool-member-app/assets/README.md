Place app icons and splash images here:

- icon.png         (1024x1024)
- adaptive-icon.png (1024x1024 foreground)
- splash.png       (~1242x2436)
- favicon.png      (48x48)

Until real assets are added, you can copy any 1024x1024 placeholder PNG into
this folder. Expo will auto-generate sized variants at build time.
