# Glenridge Pool Admin App

Cross-platform mobile app (iOS phone, iPad, Android phone, Android tablet) for
HOA pool administrators. Used at poolside or remotely to:

- Enable/disable guest and vendor pool access.
- View each guest/vendor's registered phone or NFC card credential.
- Watch the live entry log feed (~2 second refresh).
- View gate sync status and trigger manual resync.
- Mask credential identifiers by default; unmask only for super-admin role.

Built with **React Native + Expo + TypeScript**.

See `pool-admin-app/README.md` for run instructions.
